🌟 Hướng dẫn sử dụng

1. Chuyển Word sang PDF: Truy cập trang word.cloud.microsoft, tải file Word của bạn lên, chọn File → Xuất (Export) → Tải xuống dưới dạng PDF.
2. Chuẩn bị đề: Tải file PDF vừa có lên mục "Tải Đề Thi".
3. Tạo JSON: Nhấn "Copy Prompt", gửi kèm nội dung đáp án cho AI để nhận mã JSON, lưu thành file .json.
4. Ôn tập: Tải file JSON lên và nhấn "Bắt đầu làm bài"!

💡 Hướng dẫn tạo File JSON

Copy prompt dưới đây gửi cho AI (Gemini/ChatGPT) kèm theo nội dung đáp án của bạn để lấy file JSON đúng chuẩn ứng dụng.

Hãy đóng vai một công cụ trích xuất dữ liệu. Trích xuất đáp án thành JSON theo cấu trúc chính xác sau: {"abcd": {"1": 0, "2": 1}, "ds": {"1": "ddss"}, "tln": {"1": "đáp án"}}. Giải thích: Nhóm "abcd" là trắc nghiệm 4 lựa chọn (A=0, B=1, C=2, D=3). Nhóm "ds" là Đúng/Sai (d=Đúng, s=Sai, chuỗi 4 ký tự cho 4 ý a,b,c,d). Nhóm "tln" là trả lời ngắn. Các key ("1", "2"...) là số thứ tự câu trong từng phần. Chỉ xuất duy nhất mã JSON, không giải thích.
